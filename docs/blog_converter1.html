ブログをウェブサイトにする静的サイトジェネレータ

これまでここのブログは markdown で書いて github に挙げるだけという簡単な運用でした。 そろそろ何かのツールでウェブサイトに変換して公開できるようにしたいです。

欲しい機能は以下ぐらいです

  • markdown を html に変換する
    • シンタックスハイライトがほしい
    • スタイルシートを自由にしたい
  • 表紙ページの作成

ぱっと思いつくだけでも、以下の３つが有力そうです。

  • astro
  • jekyll
  • mkdocs

Awesome Static Web Site Generators のリポジトリも見つかりました。ブログの項目だけでも30個ぐらいありそうです。
